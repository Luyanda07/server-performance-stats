# Server Stats
https://github.com/Luyanda07/server-performance-stats

A simple command-line tool to view server performance stats at a glance. Available for both Linux and Windows.

## What It Shows

- OS version, uptime & load average
- CPU usage
- Memory usage (free vs used with percentage)
- Disk usage (free vs used with percentage)
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- Logged in users count

## Quick Start

### Linux

chmod +x server-stats.sh
./server-stats.sh

## Windows (PowerShell)

.\server-stats.ps1
